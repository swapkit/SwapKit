This guide covers Starknet integration with SwapKit, including wallet connections, STRK transfers, ETH operations, and cross-chain swaps.

## Overview

Starknet is a validity rollup (ZK-rollup) Layer 2 network built on Ethereum, using Cairo VM for smart contracts and native account abstraction. SwapKit provides comprehensive Starknet support through:

- **Starknet Toolbox**: Native Starknet operations with STRK and ETH tokens
- **Account Abstraction**: Native support for Starknet's account model
- **Cairo Contract Support**: Interact with Cairo-based smart contracts
- **TypedData Signing**: Sign structured data following Starknet standards
- **Multi-Call Support**: Execute multiple contract calls in a single transaction
- **Cross-Chain Swaps**: Seamless swaps with Bitcoin, Ethereum, and other chains via THORChain

## Starknet Toolbox

### Wallet Connection

Connect to Starknet using various wallet types:

```typescript
// @noErrorValidation
import { Chain, FeeOption } from "@swapkit/sdk";

await swapKit.connectKeystore([Chain.Starknet], "your mnemonic phrase");

await swapKit.connectArgent([Chain.Starknet]);

await swapKit.connectBraavos([Chain.Starknet]);

await swapKit.connectLedger([Chain.Starknet]);
```

### Address Generation and Validation

Starknet addresses are 64-character hexadecimal strings with a `0x` prefix:

```typescript
// @noErrorValidation
import { validateStarknetAddress, deriveStarknetKeys } from "@swapkit/toolboxes/starknet";

const isValidAddress = validateStarknetAddress(
  "0x04718f5a0fc34cc1af16a1cdee98ffb20c31f5cd61d6ab07201858f4287c938d"
);
console.log("Valid Starknet address:", isValidAddress);

const starknetAddress = await starknetToolbox.getAddress();
console.log("Starknet address:", starknetAddress);

const keys = await deriveStarknetKeys({
  phrase: "your mnemonic phrase",
  derivationPath: "m/44'/9004'/0'/0/0"
});

console.log("Derived keys:", {
  address: keys.address,
  publicKey: keys.publicKey
});
```

### STRK Transfers

```typescript
// @noErrorValidation
import { AssetValue } from "@swapkit/sdk";

const txHash = await swapKit.transfer({
  recipient: "0x04718f5a0fc34cc1af16a1cdee98ffb20c31f5cd61d6ab07201858f4287c938d",
  assetValue: AssetValue.from({
    chain: Chain.Starknet,
    value: "10",
  }),
  feeOptionKey: FeeOption.Fast,
});

console.log(`Starknet transaction hash: ${txHash}`);

const directTransfer = await starknetToolbox.transfer({
  recipient: "0x04718f5a0fc34cc1af16a1cdee98ffb20c31f5cd61d6ab07201858f4287c938d",
  assetValue: AssetValue.from({
    chain: Chain.Starknet,
    value: "5",
  }),
  memo: "Payment for services"
});

console.log("Direct transfer completed:", directTransfer);
```

### TypedData Signing

Sign structured data following Starknet's typed data standard:

```typescript
// @noErrorValidation

const typedData = {
  types: {
    StarkNetDomain: [
      { name: "name", type: "felt" },
      { name: "version", type: "felt" },
      { name: "chainId", type: "felt" },
    ],
    Person: [
      { name: "name", type: "felt" },
      { name: "wallet", type: "felt" },
    ],
    Mail: [
      { name: "from", type: "Person" },
      { name: "to", type: "Person" },
      { name: "contents", type: "felt" },
    ],
  },
  primaryType: "Mail",
  domain: {
    name: "Starknet Mail",
    version: "1",
    chainId: "0x534e5f4d41494e",
  },
  message: {
    from: {
      name: "Alice",
      wallet: "0x...",
    },
    to: {
      name: "Bob",
      wallet: "0x...",
    },
    contents: "Hello, Bob!",
  },
};

const signature = await starknetToolbox.signMessage(typedData);
console.log("Signature:", signature);
```

## Cross-Chain Swaps

Use Starknet as source or destination for cross-chain swaps via THORChain:

```typescript
// @noErrorValidation

const strkToBtcQuote = await swapKit.getQuote({
  sellAsset: "STRK.STRK",
  sellAmount: "100",
  buyAsset: "BTC.BTC",
  senderAddress: swapKit.getAddress(Chain.Starknet),
  recipientAddress: swapKit.getAddress(Chain.Bitcoin),
});

console.log("STRK -> BTC quote:", {
  expectedOutput: strkToBtcQuote.expectedOutput,
  fees: strkToBtcQuote.fees,
  timeEstimate: strkToBtcQuote.timeEstimate,
});

const swapTx = await swapKit.swap({
  route: strkToBtcQuote.routes[0],
  feeOptionKey: FeeOption.Fast,
});

const ethToStrkQuote = await swapKit.getQuote({
  sellAsset: "ETH.ETH",
  sellAmount: "0.1",
  buyAsset: "STRK.STRK",
  senderAddress: swapKit.getAddress(Chain.Ethereum),
  recipientAddress: swapKit.getAddress(Chain.Starknet),
});

const ethSwap = await swapKit.swap({
  route: ethToStrkQuote.routes[0],
  feeOptionKey: FeeOption.Average,
});
```

## Starknet-Specific Features

### Account Abstraction

Starknet uses native account abstraction where accounts are smart contracts:

```typescript
// @noErrorValidation

const accountClassHash = "0x01a736d6ed154502257f02b1ccdf4d9d1089f80811cd6acad48e6b6a9d1f2003";

const accountCode = await starknetToolbox.getProvider().getClassAt(
  await starknetToolbox.getAddress()
);

console.log("Account is a smart contract with code");
```

### Multi-Call Transactions

Execute multiple operations in a single transaction:

```typescript
// @noErrorValidation

const multiCallTx = async () => {
  const calls = [
    await starknetToolbox.createTransaction({
      recipient: "0xrecipient1...",
      assetValue: AssetValue.from({ chain: Chain.Starknet, value: "5" }),
    }),
    await starknetToolbox.createTransaction({
      recipient: "0xrecipient2...",
      assetValue: AssetValue.from({ chain: Chain.Starknet, value: "10" }),
    }),
  ].flat();

  const txHash = await starknetToolbox.broadcastTransaction(calls);
  console.log("Multi-call transaction:", txHash);
};
```

## Error Handling

Handle Starknet-specific errors appropriately:

```typescript
// @noErrorValidation
import { SwapKitError } from "@swapkit/sdk";

try {
  await swapKit.transfer({
    /* ... */
  });
} catch (error) {
  if (error instanceof SwapKitError) {
    switch (error.code) {
      case "toolbox_starknet_no_signer":
        console.error("No signer available - wallet not connected");
        break;
      case "toolbox_starknet_invalid_address":
        console.error("Invalid Starknet address format");
        break;
      case "toolbox_starknet_fee_estimation_failed":
        console.error("Could not estimate transaction fee:", error.info);
        break;
      case "core_transaction_invalid_recipient_address":
        console.error("Invalid recipient address:", error.info);
        break;
      default:
        console.error("Unknown Starknet error:", error);
    }
  }
}
```

## Best Practices

1. **Always validate addresses** before sending transactions:

   ```typescript
   import { validateStarknetAddress } from "@swapkit/toolboxes/starknet";

   if (!validateStarknetAddress(recipient)) {
     throw new Error("Invalid Starknet address");
   }
   ```

2. **Estimate fees** before executing transactions:

   ```typescript
   const fee = await starknetToolbox.estimateTransactionFee({
     recipient,
     assetValue,
   });

   console.log(`Transaction will cost approximately ${fee.getValue("string")} STRK`);
   ```

3. **Use appropriate derivation paths**:

   ```typescript
   const derivationPaths = {
     default: "m/44'/9004'/0'/0/0",
     account1: "m/44'/9004'/0'/0/1",
     account2: "m/44'/9004'/0'/0/2",
   };
   ```

4. **Handle account deployment**:

   Starknet accounts need to be deployed before they can receive transactions. Ensure accounts are deployed before sending funds.

5. **Consider multi-call for batch operations**:

   ```typescript
   const batchTransfer = async (recipients: Array<{address: string, amount: string}>) => {
     const calls = await Promise.all(
       recipients.map(({address, amount}) =>
         starknetToolbox.createTransaction({
           recipient: address,
           assetValue: AssetValue.from({ chain: Chain.Starknet, value: amount }),
         })
       )
     );

     return starknetToolbox.broadcastTransaction(calls.flat());
   };
   ```

## API Reference Summary

### Core Methods

- `getBalance()` - Get STRK and ETH balances
- `transfer()` - Send STRK or ETH tokens
- `getAddress()` - Get connected wallet address
- `getProvider()` - Get Starknet RPC provider

### Transaction Methods

- `createTransaction()` - Build transfer calls
- `broadcastTransaction()` - Execute calls on network
- `signAndBroadcastTransaction()` - Sign and execute in one step
- `estimateTransactionFee()` - Estimate transaction costs

### Key Management Methods

- `deriveStarknetKeys()` - Derive keys from mnemonic
- `createKeysForPath()` - Create account for derivation path
- `validateStarknetAddress()` - Validate address format

### Signing Methods

- `signMessage()` - Sign TypedData messages

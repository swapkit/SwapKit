## Key Derivation

### Derivation Path

```
m/44'/9004'/0'/0/{index}
```

- **Purpose**: `44'` (BIP44 standard)
- **Coin Type**: `9004'` (Starknet)
- **Account**: `0'`
- **Change**: `0`
- **Address Index**: `{index}` (0, 1, 2, ...)

### Key Generation Process

```typescript
// @noErrorValidation
import { deriveStarknetKeys } from "@swapkit/toolboxes/starknet";

// Step 1: Derive keys from mnemonic
const keys = await deriveStarknetKeys({
  phrase: "your twelve word mnemonic phrase here",
  derivationPath: "m/44'/9004'/0'/0/0"
});

// Step 2: Keys structure
// keys.privateKey - Ground key for signing (Stark curve)
// keys.publicKey - Public key derived from private key
// keys.address - Contract address computed from public key + class hash
```

### Address Computation

Starknet addresses are computed from:
1. Public key
2. Account class hash (e.g., OpenZeppelin)
3. Constructor calldata
4. Salt (0 for standard derivation)

```typescript
// @noErrorValidation
// Address computation formula (internal)
const address = hash.calculateContractAddressFromHash(
  publicKey,           // Public key as salt
  OZ_ACCOUNT_CLASS_HASH, // Account implementation
  constructorCallData, // CallData.compile({ publicKey })
  0                    // Deployer address (0 for UDC)
);
```

### Core Operations

```typescript
// @noErrorValidation

// Get address
const address = await toolbox.getAddress();

// Validate address
const isValid = validateStarknetAddress("0x...");

// Get balances
const balances = await toolbox.getBalance(address);

// Estimate fee
const fee = await toolbox.estimateTransactionFee({
  recipient: "0x...",
  assetValue: AssetValue.from({ chain: Chain.Starknet, value: "10" })
});

// Transfer STRK
const txHash = await toolbox.transfer({
  recipient: "0x...",
  assetValue: AssetValue.from({ chain: Chain.Starknet, value: "10" }),
  memo: "Optional memo"
});

// Sign typed data
const signature = await toolbox.signMessage(typedData);
```


### Cross-Chain Swaps

```typescript
// @noErrorValidation

// Get quote
const quote = await swapKit.getQuote({
  sellAsset: "STRK.STRK",
  sellAmount: "100",
  buyAsset: "BTC.BTC",
  senderAddress: swapKit.getAddress(Chain.Starknet),
  recipientAddress: swapKit.getAddress(Chain.Bitcoin)
});

// Execute swap
const swapTx = await swapKit.swap({
  route: quote.routes[0],
  feeOptionKey: FeeOption.Fast
});
```

## Starknet-Specific Considerations

### Account Abstraction

Starknet accounts are smart contracts, meaning:
1. Accounts must be **deployed** before receiving tokens
2. Account logic is defined by the **class hash**
3. Signatures use the **STARK curve** (not secp256k1)

### Transaction Types

1. **INVOKE** - Call contract methods (transfers, etc.)
2. **DECLARE** - Publish contract class
3. **DEPLOY_ACCOUNT** - Deploy a new account

### Fee Payment

- Fees are paid in **STRK** or **ETH**
- Fee estimation returns the maximum fee
- Actual fee may be lower

### Block Finality

- Blocks are **pending** until proven
- L1 finality requires proof submission (~hours)
- L2 finality is fast (~seconds)

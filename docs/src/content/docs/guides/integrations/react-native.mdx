## Installation

Install SwapKit and its dependencies:

<Tabs syncKey="package-manager">
  <TabItem label="bun" icon="bun">
    ```bash
    bun add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npm install @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="expo">
    ```bash
    expo install @swapkit/sdk
    ```
  </TabItem>
</Tabs>

## Required Polyfills

React Native requires several polyfills for crypto and Node.js functionality:

Core polyfills:

<Tabs syncKey="package-manager">
  <TabItem label="bun" icon="bun">
    ```bash
    bun add \
      react-native-crypto \
      react-native-randombytes \
      react-native-get-random-values \
      events \
      stream-browserify \
      readable-stream \
      buffer \
      process
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash

      react-native-crypto \
    ```

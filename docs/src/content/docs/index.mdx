## What is SwapKit?

SwapKit is a comprehensive, production-ready SDK that unifies cross-chain DeFi interactions into a single, elegant API. Supporting **20+ blockchains**, **20+ wallets**, and access to **6,000+ assets** across multiple DeFi protocols, SwapKit eliminates the complexity of building multi-chain applications.

**Built for flexibility**: SwapKit's highly modular architecture means you can use the full SDK or integrate just the pieces you need. Bring your own wallet infrastructure, implement custom signing logic, create protocol-specific plugins, or simply leverage SwapKit's quote aggregation and swap execution while managing everything else yourself. [Create Custom Plugins →](/guides/create-plugin) | [Build Wallet Adapters →](/guides/create-wallet)

### The Problem SwapKit Solves

Building cross-chain functionality today is hard. Each blockchain has its own RPC endpoints, wallet standards, fee models, and liquidity sources. Most DeFi protocols only work on a handful of networks. If you try to integrate multiple chains yourself, you quickly end up managing a patchwork of libraries, API keys, and bespoke logic.

**The challenges teams face**:
- Learning and maintaining separate SDKs for Bitcoin, Ethereum, Cosmos, Solana, and other networks
- Integrating dozens of wallet providers with incompatible signing interfaces
- Managing chain-specific quirks: different transaction formats, fee structures, and decimal precision
- Coordinating swaps across multiple DEX aggregators and bridging protocols
- Handling complex decimal conversions, gas estimation, and error states for each chain
- Building custom routing logic to find optimal swap paths across protocols

**SwapKit eliminates this complexity** with a unified interface that works consistently across all supported chains, wallets, and protocols. Write your code once, and it works everywhere.

## See SwapKit in Action

Execute cross-chain swaps in just a few lines of code:

```typescript
// @noErrorValidation
import { createSwapKit, Chain, SwapKitApi } from "@swapkit/sdk";

const swapKit = createSwapKit("your-swapkit-api-key");
await swapKit.connectKeystore([Chain.Ethereum, Chain.Bitcoin], "your mnemonic phrase");

const { routes } = await SwapKitApi.getSwapQuote({
  sellAsset: "ETH.ETH", sellAmount: "1", buyAsset: "BTC.BTC",
  sourceAddress: swapKit.getAddress(Chain.Ethereum),
  destinationAddress: swapKit.getAddress(Chain.Bitcoin),
});

const route = routes.find((r) => r.meta?.tags?.includes("RECOMMENDED"));
const txHash = await swapKit.swap({ route });
```

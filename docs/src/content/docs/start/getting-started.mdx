Install SwapKit and get configured in under 2 minutes.

## Prerequisites

Before starting, ensure you have:

- Node.js 18+ or Bun installed
- Basic knowledge of JavaScript/TypeScript
- A code editor (VS Code recommended)

## Installation

<Tabs>
  <TabItem label="bun">
    ```bash
    bun add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="pnpm">

    ```bash
    pnpm add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="npm">

## Basic Setup

### Simple

```typescript
// @noErrorValidation
import { createSwapKit } from "@swapkit/sdk";

const swapKit = createSwapKit();
```

### With API Key

```typescript
// @noErrorValidation
import { createSwapKit } from "@swapkit/sdk";

const swapKit = createSwapKit("your-swapkit-api-key");
```

### Advanced

Full configuration with custom RPC URLs and API keys. See [Configuration Guide](/start/configuration) for details:

```typescript
// @noErrorValidation

import { createSwapKit } from "@swapkit/sdk";

const swapKit = createSwapKit({
  config: {
    apiKeys: {
      swapKit: "your-swapkit-key",
      walletConnectProjectId: "your-project-id",
      blockchair: "your-blockchair-key",
    },
    rpcUrls: {
      ETH: "https://eth-mainnet.g.alchemy.com/v2/your-key",
      AVAX: "https://api.avax.network/ext/bc/C/rpc",
    },
  },
});
```
